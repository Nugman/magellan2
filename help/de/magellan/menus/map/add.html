Auswahl hinzufügen

Dieser Menüpunkt öffnet einen Dialog und fragt nach einer Selection-Datei, die mit Auswahl speichern erstellt wurde. So kann man eine Auswahl wieder herstellen. Diese Funktion lässt bereits ausgewählte Regionen ausgewählt.

Hinweis:
Speichert man einen CR mit ausgewählten Regionen, so wird diese Information im CR hinterlegt. Exportiert man einen Report im Kompatibilitätsmodus, so werden diese Informationen nicht hinterlegt.
