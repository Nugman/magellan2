## Neueste Meldungen

Eine ganze Menge eher kleinerer, aber merkbarer Änderungen macht diese Version aus.

Die wichtigsten Änderungen:

  * Magellan braucht jetzt Java 11. Dies ist derzeit die einzige Javaversion, die noch ausreichend mit Updates versorgt wird
  * Die ARR und ATR-Eigenschaften wurden poliert. Eine Erklärung zum ATR und ARR gibt es bald im Eresseawiki (https://wiki.eressea.de/index.php/Magellan).
  * Unterstützung für Konvois (GIB xyz 1 SCHIFF)
  * E3CommandParser benutzt jetzt die JShell und ist dadurch viel, viel schneller. Der E3CommandParser erlaubt es, Befehle weitgehend zu programmieren.
  * Die Befehlserkennung und -verarbeitung wurde in vielen Punkten verbessert. ECheck ist praktisch überflüssig und durch "Offene Probleme" ersetzt.
  * Weitere Regelupdates (Galeonen, Drachenschiffe, !@Befehle, BEZAHLE, BEFÖRDERE...) wurden implementiert
  * Das Plugin "BattleAnalyzer" stellt schlachten etwas übersichtlicher dar.
  * Die Talentstatistik in der Parteienstatistik wurde verbessert.
  * Unterstützung für andere Spiele (E3, Atlantis, Drachensgrab, ...)
  * Viele, viele weitere Bugfixes und Verbesserungen an der Bedienbarkeit ...

Für eine Liste mit allen Änderungen siehe CHANGELOG.txt.

## Release Notes

There were a bunch of mostly minor, but notable changes leading up to this version.

The most important changes (for a complete list of changes see CHANGELOG.txt):

  * Magellan now requires Java 11. This is currently the only version that is going to provide regular updates.
  * The ARR and ARR proerties were polished. For an explanation visit the Magellan page in the Eressea wiki (https://wiki.eressea.de/index.php/Magellan).
  * Support for convoys (GIVE xyz 1 SHIP).
  * E3CommandParser uses JShell now, making it much faster. E3CommandParser allows you to automate your orders.
  * Order detection and processing was improved in many parts. ECheck ist practically unnecessary and replaced by the "Open Tasks".
  * Further rules updates implemented (galleons, dragon ships, !@ orders, PAY NOT, PROMOTE, ...).
  * The "BattleAnalyzer" plugin displays battles a bit more comprehensively.
  * The skill statistics of the faction statistics has been improved.
  * Support for other games (E3, Atlantis, Drachensgrab, ...)
  * Many, many further bug fixes and improvements ...
