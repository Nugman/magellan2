This is the editor environment of Extended Commands.

This text attempts to clarify the complex possibilities of this plugin. This editor is based on JShell and therefore similar to Java. There are classes and methods. The variables are type safe. See the introduction into ExtendedCommands.

At the moment you are working on the library for all units and containers. The following global variables are available:

  • world - the most recently loaded report and all information connected with it (islands, regions, factions, etc.)
  • helper an auxiliary class with handy functions

To be able to use additional classes from Java or Magellan, these have to be imported. This happens for example as follows:

import magellan.library.*;

int getHorses(Region region) {
  return region.getHorses();
}
